          To build and run this NSH Task Demo‚Ä¶

          1. Install the Build Prerequisites, skip the RISC-V Toolchain‚Ä¶

            ‚ÄúInstall Prerequisites‚Äù

          2. Download the ARM64 Toolchain for AArch64 Bare-Metal Target aarch64-none-elf

            Arm GNU Toolchain Downloads

            (Skip the section for Beta Releases)

          3. Add the downloaded toolchain to the PATH Environment Variable‚Ä¶

            gcc-arm-...-aarch64-none-elf/bin

            Check the ARM64 Toolchain‚Ä¶

            aarch64-none-elf-gcc -v

          4. Download QEMU Machine Emulator‚Ä¶

            ‚ÄúDownload QEMU‚Äù

          5. Download NuttX‚Ä¶

            mkdir nuttx
cd nuttx
git clone https://github.com/apache/nuttx nuttx
git clone https://github.com/apache/nuttx-apps apps
cd nuttx

          6. Add nshtask to our NuttX Project‚Ä¶

            pushd ../apps/examples
git submodule add https://github.com/lupyuen/nshtask
popd

          7. Configure our NuttX Project‚Ä¶

            tools/configure.sh -l qemu-armv8a:nsh
make menuconfig

          8. In ‚ÄúApplication Configuration > Examples‚Äù

            Enable ‚ÄúNSH Task Demo‚Äù

          9. Optional: If we wish to start nshtask when NuttX boots‚Ä¶

            In ‚ÄúRTOS Features > Tasks and Scheduling‚Äù

            Set ‚ÄúApplication entry point‚Äù to nshtask_main

            Set ‚ÄúApplication entry name‚Äù to nshtask_main

          10. Save the configuration and exit menuconfig

          11. Build NuttX‚Ä¶

            make

          12. Run NuttX with QEMU‚Ä¶

            qemu-system-aarch64 -cpu cortex-a53 -nographic \
  -machine virt,virtualization=on,gic-version=3 \
  -net none -chardev stdio,id=con,mux=on -serial chardev:con \
  -mon chardev=con,mode=readline -kernel ./nuttx

          13. At the NSH Prompt, enter this to run the demo‚Ä¶

            nshtask

            When we‚Äôre done, press Ctrl-C to quit QEMU.
